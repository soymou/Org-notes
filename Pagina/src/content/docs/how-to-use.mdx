This guide explains how to write notes in Org-mode for this Astro
Starlight site. This file is hidden from the sidebar but accessible via
the current URL.

# Basic Formatting

Write standard Org-mode syntax:

- **Bold text**
- *Italic text*
- <u>Underlined text</u>
- `Code snippets`

# Mathematical Equations

- ****Inline math****: Use single dollar signs: $E = mc^2$.
- ****Display math****: Use double dollar signs on their own lines:

$$
\sum_{i=1}^{n} i = \frac{n(n+1)}{2}
$$
